#### serial.Char

Send a single-byte character.

#####  Syntax

    
    
    serial.Char(bytechar)

  * ** bytechar ** \- ASCII character value to send. 

Returns no values.

#####  Description

Use this command to send data to an external serial device.

#####  Example

    
    
    PUB Main
        serial.Start
     
        serial.Char(34)
        serial.Char("H")
     
        serial.Stop
