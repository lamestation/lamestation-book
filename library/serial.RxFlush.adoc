#### serial.RxFlush

Empty the receive buffer.

#####  Syntax

    
    
    serial.RxFlush

This command takes no arguments and returns no values.

#####  Description

Empty the receive buffer.

#####  Example

    
    
    PUB Main
        serial.Start
        serial.RxFlush
        serial.Clear
     
        serial.Str(String("Blah blah blah blah"))
     
        serial.Stop
