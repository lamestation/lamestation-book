#### serial.Stop

Stop serial communication and free the cog.

#####  Syntax

    
    
    serial.Stop

This command takes no parameters and returns no values.

#####  Description

Stop serial communication and free the cog.

#####  Example

    
    
    PUB Main
        serial.Start
     
        ' some commands
     
        serial.Stop
